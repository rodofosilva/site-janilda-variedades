# 🛍️ Janilda Variedades

Landing page desenvolvida para a loja **Janilda Variedades**, com foco em divulgação de produtos e aumento do contato com clientes através do WhatsApp.

---

## 🎯 Objetivo do Projeto

Criar um site moderno, simples e funcional para:

- Divulgar os produtos da loja  
- Atrair novos clientes  
- Facilitar o contato direto via WhatsApp  
- Melhorar a presença digital da loja  

---

## 🧠 Levantamento de Requisitos

O projeto foi desenvolvido com base nas necessidades da cliente, que possui uma loja com mais de 15 anos de atuação.

Principais pontos levantados:

- Exibir variedade de produtos (roupas, joias, perfumes, etc.)  
- Site simples e fácil de usar  
- Foco em acesso via celular  
- Contato rápido com o cliente  
- Destaque visual para os produtos  

---

## 🛠️ Tecnologias Utilizadas

- HTML5  
- CSS3 (com animações e responsividade)  
- JavaScript (interatividade e galeria dinâmica)  

---

## ⚙️ Funcionalidades

- ✅ Página totalmente responsiva (mobile e desktop)  
- ✅ Galeria de produtos por categoria (modal dinâmico)  
- ✅ Efeito de animação ao rolar a página  
- ✅ Efeito hover com zoom nas imagens  
- ✅ Botão flutuante de WhatsApp  
- ✅ Botões de contato direto (WhatsApp e Instagram)  
- ✅ Organização visual moderna e intuitiva
